Connect Your Operating System
SensorModel exposes a local WebSocket + REST API. Connect any OS, SCADA, PLC, or data historian in minutes.
Python (Most common)
# Install: pip install sensormodel-client import sensormodel # Connect to SensorModel client = sensormodel.Client("ws://localhost:8765") # Stream real-time predictions for prediction in client.stream("bearing_vibration"): print(prediction.rul_hours) # Remaining Useful Life print(prediction.anomaly_score) # 0.0 - 1.0 print(prediction.next_value) # Forecasted value print(prediction.confidence) # Model confidence %
REST API (Any language)
# POST sensor reading, get prediction curl -X POST http://localhost:8080/api/predict \ -H "Content-Type: application/json" \ -d '{ "sensor_id": "motor_current_01", "value": 18.4, "timestamp": "2026-06-15T10:32:00Z", "asset": "centrifugal_pump", "unit": "A" }' # Response: { "rul_hours": 847, "anomaly_score": 0.12, "predicted_next": 18.6, "confidence": 0.94, "alert_level": "NOMINAL", "features": {"rms": 18.4, "kurtosis": 2.1, "trend": 0.02} }
Node.js / JavaScript
const ws = new WebSocket('ws://localhost:8765'); ws.onmessage = (event) => { const pred = JSON.parse(event.data); console.log(`RUL: ${pred.rul_hours}h | Alert: ${pred.alert}`); };
